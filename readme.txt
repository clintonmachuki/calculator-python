Simple Python Calculator
This is a simple Python calculator with a graphical user interface (GUI) built using the Tkinter library.

Features
Basic arithmetic operations: addition, subtraction, multiplication, division
Additional mathematical functions: square root, logarithm, sine, cosine, tangent
Error handling for invalid inputs
Clear button to reset the input field
Detailed README file for easy understanding
Prerequisites
Python 3.x installed on your system
